# Wayli architecture and roadmap

## Where Wayli is going

Wayli is not intended to be a single calculator website.

The long-term opportunity is a **consumer decision platform** that helps UK households make better financial choices using practical tools, plain-English guidance and scenario comparisons.

Current and future tools may include:

- Student loan overpay vs invest
- Mortgage overpay vs invest
- Repair or replace car
- Rent vs buy
- EV switch tool
- Childcare hours / affordability tool
- Savings allocation decisions
- Insurance excess vs premium trade-offs

That means Wayli should be built as a **shared platform**, not as disconnected one-off tools.

---

# Core product principle

Every Wayli tool should answer the same user question:

> “What is likely to be the strongest decision for my situation?”

Not generic education.  
Not endless spreadsheets.  
A clear, personalised comparison.

---

# The future Wayli model

## Move away from:

One custom Python file per tool  
One-off front-end pages  
Different layouts for every calculator

## Move toward:

A reusable **Decision Engine Framework**

Where each tool plugs into common inputs, outputs, ranking logic and presentation.
